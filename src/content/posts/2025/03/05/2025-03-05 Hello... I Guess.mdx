จริงๆคืออยากลองทำแนว Blog มานานแล้วเพราะว่าเป็นคนขี้บ่น แล้วก็พบว่า Facebook เป็นสถานที่ที่แปลกเหมือนเราพูดมากอยู่คนเดียวแต่แทบไม่มีคนบ่นอะไรเท่าไร นานๆทีก็เลยอยากเขียนอะไรเก็บไว้ดูเองมั้งละมั้ง? คือไหนๆแล้วก็รู้สึกว่าอยากลองลงมือทำโน้นนี้ดูบ้าง

---
***ทำไมต้องแยก Blog ออกมานะ?***

นั้นสิ.....

คือตอนแรกในหัวคืออยากทำเวบ ออกมา 3 อัน คือ

1. อยากทำเวบบอร์ดจะได้เอาไว้เก็บบทความให้เป็นที่เป็นทาง
2. อยากทำ blog เอาไว้เขียนบ่นเรื่อยเปื่อยแบบไม่มีใครอ่าน (แต่ถ้ามีคงดีใจตายเลย)
3. ทำ web app เอาไว้ filter เกมสำหรับแนะนำคน

ซึ่ง จริงๆ ***1*** เนี่ยเราใช้ flarum ทำ webboard แล้วมันมี plugin ที่สามารถใส่ blog เข้าไปได้เลยเว้ยคือรวม ***2*** ไปด้วยกันได้ทันที แต่มันก็มีคำถามแบบคนคิดมากอยู่ในใจ

- อย่างแรกคือถ้าเรามี blog อยู่คนเดียวมันก็จะดูเป็นเพจส่วนตัวไปปล่าวว่ะ? คือตอนนี้มันไม่มีคนมาใช้แต่ถ้าในระยะยาวล่ะ?
- อย่างที่สองคือพอมันเป็นบอร์ดหัวข้อเฉพาะทางมันจะไปใส่เทคใส่บ่นไร้สาระอะไรก็คงแปลกๆ
- อย่างที่สามก็นั้นแหละอยากลองวิชาท่าอื่นด้วย แล้วพอดีเจอ host ถูกเว่อร์เดือนละ 1 usd ก็เลยลองดูหน่อยละกัน
- อย่างที่สี่คือมีบทเรียนจากตอน wordpress พื้นที่เต็ม ย้ายบทความออกมายากมากเพราะไม่ได้ทำเป็น markdown เอาไว้ รอบนี้ก็เลยคิดว่าอยากได้ระบบที่มัน lightweight มากๆย้ายง่ายๆดู

---
blog เนี่ยตอนแรกจะใช้ astro แล้ว upload ขี้นเป็น static web เอาแต่ว่ากลัวว่าตัวเองจะขี้เกียจทำแก้ ทำ build ไรงี้ เลยคิดว่าไม่เอาดีกว่า แต่ยังไงถ้าเตรียมบทความเป็น markdown ไว้ก็คงย้ายง่ายแหละเพราะไม่ต้องจัด format ใหม่
